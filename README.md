# nlw04-Rockseat

## Backend - api:

### tecnologias utilizadas
- Linguagem: Typescript/NodeJS
- Framework/Lib: Express
- ORM: Typeorm
- Banco SQL: Sqlite

### instalar as dependencias
```bash
yarn add
```
### rodar backend
```bash
yarn dev
```

### rodar migration
```bash
yarn typeorm migration:run
```

### reverter migration
```bash
yarn typeorm migration:revert
```
